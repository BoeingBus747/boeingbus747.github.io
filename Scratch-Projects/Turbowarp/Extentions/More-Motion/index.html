More Motion

"More Motion" is an expantion of the Motion Section of the Code Category, just like most other Turbowarp extentions it will not run on normal Scratch.

Its Blocks

  • Change x (amount), y (amount)
  • Point towards x (input), y (input)
  • REPORTER: Rotation Style
  • Manually Fence
  • Move (INPUT) steps towards x: (input), y: (input)
  • Move (INPUT)% of the way to x: (input), y: (input)
  • REPORTER: Direction to x: (input), y: (input)
  • REPORTER: Distance from x: (input), y: (input)
  • REPORTER: Sprite (Width/Height/Costume Width/Costume Heigh)
  • BOOLEAN: Touching X(input), Y(input)?
  • BOOLEAN: Touching rectangle X1(Input), Y1(Input), X2(Input), Y2(Input)?
