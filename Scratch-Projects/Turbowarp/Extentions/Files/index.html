Turbowarp's Files Extention

Turbowarp's Files Extention is an extention that can be added to the TurboWarp Editor but NOT the Normal Scratch Editor as it is incompatible with it.

Its Blocks

  • Open File as (Text OR data:URL)
  • Open a (INPUT) file as (Text OR data:URL)
  • Download (INPUT) as (INPUT)
  • Download URL (INPUT) as (INPUT)
  • Set open file selector mode to (Show Modal, Open Selector Immediately, OR Only Show Selector (unreliable))
